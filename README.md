# FUCSS next wrapper

#### install
`npm install --save next-fucss`  

#### origin
https://fucss.com

### React-Static setup

```npm instal --save next-fucss```  
## Add to static.config.js plugin
generated style file is **/public/style.css**  
```javascript
    plugins: [
      require.resolve('next-fucss')
    ]
```
